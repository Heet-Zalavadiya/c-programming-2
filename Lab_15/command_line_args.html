Command Line Arguments in C

What are Command Line Arguments?

Command line arguments are values passed to a C program from the operating system's command line (or via the IDE) at the time of execution. They allow you to control the behavior of your program from the outside without hardcoding values or requiring user input via scanf.

1. The Syntax

To accept command-line arguments, the main() function must be defined with two specific arguments: argc and argv.

int main(int argc, char *argv[]) {
    // Program logic here
    return 0;
}

Understanding the Parameters:

  • int argc (Argument Count): This integer stores the total number of arguments passed to the program. The program name itself is always counted as the first argument, so argc is always at least 1.
  • char *argv[] (Argument Vector): This is an array of string pointers. Each element points to a command-line argument passed.
    • argv[0] points to the name of the program/executable.
    • argv[1] points to the first argument provided by the user.
    • argv[n] points to the nth argument.
    • argv[argc] is always a NULL pointer.

2. Example Program

Here is a basic C program that prints all the arguments passed to it.

#include <stdio.h>

int main(int argc, char *argv[]) {
    int i;
    
    printf("Total arguments passed: %d\n", argc);
    
    // Loop through the argv array to print each argument
    for(i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    
    return 0;
}

3. How to Pass Arguments in Dev-C++

Normally, command line arguments are passed using a terminal or command prompt (e.g., program.exe arg1 arg2). However, Dev-C++ provides a built-in way to simulate this directly within the IDE.

  • Step 1: Write and Save Copy the example code above into a new Dev-C++ source file. Save it as cmd_args.c.
  • Step 2: Compile the Program Press F9 (or go to Execute -> Compile) to compile your code into an executable file. This must be done before passing parameters.
  • Step 3: Open Parameters Menu Go to the top menu bar, click on Execute, and then select Parameters... from the dropdown menu.
  • Step 4: Enter Your Arguments A dialog box will appear. In the text field labeled "Parameters to pass to your program", type your arguments separated by spaces.
    Example: apple banana 123
    Click OK.
  • Step 5: Run the Program Press F10 (or go to Execute -> Run) to run the program.

Expected Output based on Step 4:

Total arguments passed: 4
Argument 0: C:\Users\YourName\Documents\cmd_args.exe
Argument 1: apple
Argument 2: banana
Argument 3: 123
Important Note on Data Types:

All command-line arguments are read as strings (character arrays). If you pass a number like 100, it is treated as the string "100". To perform math operations on numerical arguments, you must convert them to integers or floats using functions like atoi() (ASCII to Integer) or atof() (ASCII to Float) from the <stdlib.h> library.
